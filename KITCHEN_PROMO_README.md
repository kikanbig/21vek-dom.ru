# Акция "Скидка 12% на все кухни"

## Что было создано

### 1. Изображения
- **Баннер для карусели**: `src/assets/promos/kitchen-promo-banner.png`
  - Горизонтальный формат для главной карусели на лендинге
  - Отображается на главной странице в компоненте `PromoBanner`

- **Плитка для акций**: `src/assets/promos/kitchen-promo-tile.png`
  - Квадратный формат для сетки акций
  - Отображается в компоненте `PromoGrid` на главной странице
  - Также отображается на странице `/promos` (все акции)

### 2. Страница акции
- **Путь**: `/promos/kitchen`
- **Компонент**: `src/pages/promos/KitchenPromo.tsx`
- **Содержимое**:
  - Большой баннер с изображением
  - Текст акции (из рассылки)
  - Блок "Приходите к нам в магазин" с:
    - Адресом магазина
    - Режимом работы
    - Телефоном
    - Кнопкой "Построить маршрут" (ссылка на Яндекс.Карты)
    - Кнопкой "Смотреть каталог кухонь" (ссылка на `/shop/Мебель/Кухни`)
  - Условия акции

### 3. Обновленные компоненты

#### PromoBanner.tsx
- Добавлен баннер акции на кухни в начало карусели
- Баннер автоматически прокручивается вместе с другими

#### PromoGrid.tsx
- Добавлена плитка акции на кухни в начало сетки
- Плитка отображается на главной странице (первые 6 акций)
- Плитка также отображается на странице `/promos` (все акции)

#### App.tsx
- Добавлен маршрут `/promos/kitchen` для страницы акции

#### Promos.tsx
- Добавлена плитка акции на кухни в список всех акций
- Для акции на кухни убран префикс "Сет" в заголовке

### 4. Email-рассылка
- **Файл**: `email-kitchen-promo.txt`
- Содержит текст для email-рассылки с кнопкой CTA

## Как работает

1. **Главная страница**:
   - Баннер в карусели (первый слайд)
   - Плитка в сетке акций (первая плитка)

2. **Страница всех акций** (`/promos`):
   - Плитка акции на кухни (первая в списке)

3. **Клик на баннер или плитку**:
   - Открывается страница `/promos/kitchen`
   - Страница содержит полное описание акции
   - Два CTA: "Построить маршрут" и "Смотреть каталог кухонь"
